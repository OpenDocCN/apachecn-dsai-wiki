## 下载

### Docker

```
docker pull apachecn0/flygon-dsai-wiki
docker run -tid -p <port>:80 apachecn0/flygon-dsai-wiki
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-dsai-wiki
flygon-dsai-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
